目录说明：
==========
这个目录下的文件是用于EasyWebServer测试演示的

1. 运行 EasyWebSvr.exe 后
2. 点右键菜单--设置
3. 把主目录设置为本Demo目录，填好端口（比如80），确定
4. 点右键菜单--启动服务器
3. 点右键菜单--浏览主页，就可以在浏览器中看到效果了


文件列表：
=========
1. index.htm    // 主网页
2. hello.exe    // CGI HelloWorld测试程序
3. viewmsg.exe  // CGI 留言本测试程序
4. msg.dat      // CGI 留言本测试程序的数据文件
5. hello.dll    // ISAPI HelloWorld测试程序
6. hello.php    // PHP HelloWorld的测试脚本
7. phpinfo.php  // PHP Info的测试脚本
8. hello.asp    // ASP HelloWorld测试脚本
9. hello.pl     // Perl HelloWorld测试脚本
10. readme.txt  // 本readme文件
